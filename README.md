# Numerical Algorithms in C

## 概要 (Overview)
大学の数理科学科における講義および自習で実装した、数値計算アルゴリズムのC言語コード集です。
数学的なロジックをプログラミング言語（C言語）で実装し、動作検証を行いました。

## 実装したアルゴリズム (Implemented Algorithms)
* **newton.c**: ニュートン法による高次方程式の解法
* **euler.c**: オイラー法による微分方程式の数値解法
* **matrix_calc.c**: ガウスの消去法を用いた連立一次方程式の解法
